# Release Notes – Version 0.5.0 (versionCode 5)

Für die Play Console („Was ist neu?"). Deutsch, ≤ 500 Zeichen.

## Verwendet (passt ins 500-Zeichen-Limit)

```
• Fortschritt sichtbar gemacht: Bestwert-Kacheln, ein Verlaufs-Diagramm pro Übung und eine Trainings-Heatmap mit Wochen-Streak zeigen auf einen Blick, wie du dich steigerst.
• Aufgeräumter Import: Trainingspläne werden jetzt schon beim Einfügen automatisch geprüft – ein Tipp auf „Plan übernehmen" genügt.
• Zeit-Übungen: Läuft der Timer ab, zählt eine Referenzuhr weiter, falls du länger durchhältst – die tatsächliche Zeit lässt sich exakt übernehmen.
```
